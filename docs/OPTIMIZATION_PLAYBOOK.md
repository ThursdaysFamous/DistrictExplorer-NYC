# Optimization Playbook — pointer

The **master lives in the Chicago repo** (the reference implementation), at
[`DistrictExplorer-CHI/docs/OPTIMIZATION_PLAYBOOK.md`](https://github.com/ThursdaysFamous/DistrictExplorer-CHI/blob/main/docs/OPTIMIZATION_PLAYBOOK.md).

This fork deliberately carries no copy: the two copies drifted once already —
the file that used to live here was a byte-identical snapshot of a past CHI
revision, silently falling behind as the master gained new measured tasks and
recording none of NYC's own work. Optimization tasks are engine work: measure
in the Chicago repo, land there first, and arrive here as ported engine diffs
per `docs/ENGINE_SYNC.md` (port the diff, not the prompt).

NYC-side measurements worth keeping belong in the master's task entries; NYC's
port history is archived at CHI
[`docs/archive/METRO_EXPANSION_NYC.md`](https://github.com/ThursdaysFamous/DistrictExplorer-CHI/blob/main/docs/archive/METRO_EXPANSION_NYC.md).
